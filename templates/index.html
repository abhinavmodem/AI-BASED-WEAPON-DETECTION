CCTV ANALYTICS

Comprehensive CCTV Weapon Detection System Overview

The CCTV Weapon Detection System is a comprehensive solution designed to enhance security by automatically identifying and annotating weapons in live video feeds. The system consists of several key components: data collection and preprocessing involving a diverse dataset, YOLO v8 training for object detection, integration with Robo-flow API for model management, a Flask backend for webcam integration, and email generation to alert nearby police stations. This system showcases the power of YOLO v8's object detection capabilities and leverages Robo-flow's data management tools. With its web interface, this system efficiently streams annotated video frames and sends instant email notifications to law enforcement, making it a vital tool for enhancing public safety.

Main Steps

Model Training

Develop and train YOLOv8 model for object detection, with a focus on weapon classification and feature extraction.

System Integration

Integrate Roboflow API, create Flask backend for real-time object detection, email notifications, and law enforcement communication.

Web Interface

Design a user-friendly web interface using HTML, CSS, JS, and Flask to interact with the CCTV analytics system and visualize results.
